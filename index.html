Инструкция по использованию:

  1. В поле "Целевое среднее урона" укажите требуемый средний урон для 3-й отметки на вашем танке (например, 3200).
  2. Вводите урон за каждый бой в соответствующую строку таблицы.
  3. Система автоматически подсчитает статистику и покажет, сколько осталось набрать урона.
  4. Зелёным выделяются бои, где урон выше среднего, красным - где ниже.
  5. Данные сохраняются автоматически в вашем браузере.
  6. Вы можете экспортировать данные для переноса на другое устройство или импортировать ранее сохранённые данные.
