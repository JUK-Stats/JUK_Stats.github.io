Этот сайт поможет вам отслеживать прогресс в наборе урона для получения отметок.

  • Введите урон для каждого боя в соответствующие поля таблицы
  • Если у вас что-то не отображается, то просто обновите страницу
  • Установите целевое среднее значение среднего урона на нудную вам отметку, сайт сам всё посчитает
  • Следите за статистикой и прогрессом под таблицей
  • Используйте кнопки для сброса, экспорта и импорта данных (лучше использовать Json, ведь только его можно импортировать на сайт)
  • Нажмите на кружки для изменения градиента на сайте
  • Катаете 100 боёв, вписывая урон каждого боя, 101 бой вы уже вписываете в поле первого боя, 102 второго боя и т.д.
